将要毕业的我：我为什么喜欢程序员这一行？

是的，我就要毕业了，毕业直接工作去了。 目前说来，我觉得自己很喜欢程序员这一行，当然有很多喜欢的理由，但是我目前还仅仅是个学生，大三暑假两个月的企业实习经历就足矣让我体会到在学校里有多井底之蛙，所以，这些想法里面多少有些幼稚的想法，于是我给标题加上了“将要毕业的我”，希望多年以后的我能给现在的自己分析分析这些想法，同时希望现在留下的这些文字能让多年后的自己拾起一些初心。 先从我选择就业的心路历程开始吧。 我选择就业的心路历程 家里（县城）很多亲戚会劝我读研，他们大多在国企工作，提到本科生升职会比硕士生慢，本科生 5 年硕士生 3 年这样子。我不会听他们“头头是道”的建议，但我相信他们说的情况，他们的行业确实存在很严重的学历歧视。 很多高中同学也跟我说读个研会更好找工作，听他们的描述，确实给我不读研只能在工厂里浪费青春的感觉，归根结底还是学历歧视。在他们的大学里（不乏 985 211），考研几乎是大势所趋。所幸，在我的大学，考研并不是一件理所应当的事情，甚至还是一件稍微有些丢人的事情，因为校内氛围大概是这样的：升学这块大家觉得出国是最好的，保研人（保本校的）很多会说自己摆烂保研了。这下好了，考研人连摆烂都不敢乱说了。学校对考研也不重视，几乎没有支持，所以考研的人基本完全靠自己，对自己的认同感会比较低（之前我看到有人说在这考研，就是狗都不如🤣），而且考研本就是个艰难的过程，在这样的情况下确实不容易坚持下去。 所以出国是校内公认的最好选择，而在计算机系，就业也是主流的选择，因为往届有很多学长去到大厂。所以在就业也是个不错的选项并且读研不是理所应当的氛围下，我觉得究竟读不读研很简单：喜欢科研就读，不喜欢就别读。 系里每个人在大三大四都会给安排进实验室，有牛逼的人能发文章，而我选择的导师虽然教会我很多的人生道理和技术知识，但却没让我对科研有哪怕基本的理解，总而言之，我没有喜欢上科研。反而是大三暑假去到企业实习让我大涨见识，给我一种学校里开发的东西都是玩具的感觉，我现在觉得事实确实如此，因为学校里的项目大多只管做出来而不需要维护，所以高并发、高可用这样的东西要么不考虑要么就是纸上谈兵，甚至连打 log 都有很多人没有习惯，也正常，又不需要排查线上问题，大不了 terminate 再打断点 debug 呗😜。 所以，我选择去到企业里面。 我现在确实完全不想读研，觉得工作是最好的选择，但是这多少会是因为我要读研得考研，会比较苦而且有风险，如果能够保研，我应该也会无脑选择保研。选择保研倒完全不是因为觉得提高学历牛逼或者觉得读研有什么好处，其实就是临近毕业的我现在的心态：不想离开学校，想在学校多待一段时间。这真是因为学校有什么值得留恋的吗？不是，只是对于未知的恐惧罢了，离开学校后会怎样呢？只能是边担心边期待。 我觉得，自己在寻找出路的问题上，并不是在寻找最优解，而是找一个说得过去的解，给自己一个交代，也给朋友家人一个交代。所以拿到 offer 之后，觉得已经很不错了，后面也没有再认真找更好的。反正，我不知道什么是最优解。是薪资福利吗？梦寐以求的腾讯开始裁员了，去年毕业的一个很优秀的学长被裁了；是发展前景吗？我下判断比较谨慎，不能像一些人一样把某个行业为什么有前景说得头头是道，我觉得这就是撞运气，完全体现不了判断力；那是什么呢？我不知道，我想可能是自己的直觉，虽然这很盲目，但是在根据实际条件把坑的选项去掉后，根据自己直觉选答案就够了。 我喜欢程序员这一行的理由 学历歧视不严重，能力为主 上面我说到，很多行业都有学历歧视，而程序员这一行据我目前的体会，学历歧视并不严重，能力是主要的。 一方面网上的招聘信息和程序员的交流让我觉得这一行能力最重要。只要不被卡简历，在简历上以及面试的时候体现自己的能力，对方没有因为学历不招你的理由。有腾讯的学长对二本院校的实习生赞赏有加。 另一方面是实习的观察。由于早早想好了要就业并且实习有可能转正，那我肯定要了解下实习的企业里学历歧视严不严重，如果严重，那我估计真得权衡看看要不要读个研。所以实习的时候我有特别去了解企业内部有没有学历歧视的现象存在。而据我的观察，是没有的，问了组里不同工作年限的同事，工作两年的本科生说没有，工作八年的本科生也说没有甚至本科生还有些年龄优势。同时还了解到有工作两年的中大的管理学硕士职级和工作两年的广工本科生一样，组里还有个工作四年的数学硕士由于不太会写代码职级也两年的本科生一样，感觉应该都是基于所谓创造的价值来判断人，而不是通过本科生硕士生这样片面的指标。基于这些观察，我觉得至少可以判断：在职级不是特别高的情况下，学历不会影响升迁进程，而到比较高的级别，学历是否会成为比较条件，我就不清楚了，得进企业再观察，况且能不能到那一步还是个问题呢。所以我后续考虑转正的时候，学历歧视就不用考虑了，也不用听亲戚说什么有个研究生升职更容易这样的说法，一句我问里面的人他们说还好这些人就沉默了。。。 看到这一行很多硕士生拿到 offer 一比会说硕白读，我觉得这才是正常的，可以考虑到读研的时间成本给硕士生开高一些，但如果要高很多，那完全不现实，我完全不觉得混过三年硕士的人会比本科生厉害多少（主要是也见过一些很菜的硕士）。当然，对于硕士阶段做出很厉害东西的大佬，那企业应该给他们开好很多的条件。所以，我跟我保研的舍友说，硕士身份本身不会帮你找到更好的工作，但你在硕士三年期间 提升的能力以及做出的东西可以帮助你找到更好的工作，千万不要有我是硕士就应该给我多些的想法，努力在课题组干活，提升自己的能力。 我觉得，学历歧视就是内卷的产物，人太多了，加上工作对能力的区分度不高，就搞些这种指标来筛选。 分享精神 解决问题的精神

有梗的线代老师——带带大家刚

搬运提要 这篇文章是我大一下学期期末考试周写的，当时放在坚果云，最近翻到，特搬运到博客上，记录我青春无知而又最好的年华。搬运日期：2023-05-29。 说起来，我很多对生活的探索，都是在考试周，即“除了复习考试之外干什么效率都很高”，虽然这无疑是拖延症，但是这些探索相比于狗屁的分数而言，是重要百倍千倍的。我的创造力在考试周这样的环境下最容易被激发：没有咄咄逼人的 DDL、没有需要打的卡、时间自由安排。 以下是正文。 正文 我想估计也不会有人有机会看看这个，但是其实这就像嵌在 Linux 内核代码中的句子一样，充满着浪漫主义情怀，或者更准确的说，是一种得意，或者又是一种想他人知又不想他人知的纯美小心思。 我的线代老师——杨佳刚 如果让我来回忆我的大一（事实上我还没结束大一，这让这份回忆比较准确但是很做作），我一定会觉得最秀的老师是杨佳刚，他的竞争对手有：java1A：何明昕、赵耀；高数：吴纪桃、杨柳青；大物：吴文政、鲁大为。与这些竞争对手相比，他绝对算是奇葩一朵了，比较搞耍的行为包括但不限于以下几个： 一、吹那个写《heroes in my heart》的北大校友 他说他们当年的北大 bbs 上，有一个数学系的人发表了一系列帖子，关于数学家的。我听上去老有种传奇的感觉，幻想我们这辈也能有这样的传奇人物搞出点让人兴奋的东西来。 我当时兴高采烈地去搜了这个看了看，对里面的第一段话印象深刻： 多年以前，我有一个很宏伟的计划，打算写一本厚厚的书。 这本书有三部，第一部写那些数学牛人们的传奇动人荒诞不经的轶事，第二部充满着历史上最最经典的定理最最美妙的证明，第三部去真实的纪录北大数学的这群烂人，写他们那脏乱的宿舍和芜杂的生活。 这一直是一个理想，直到我动手写这些文字的时候，我知道，这将永远是一个美好的梦。所以，这里只是那个计划的一小部分，讲述的是那些虔诚的人做过的虔诚的事。 我印象最深的是”去真实的记录北大数学的这群烂人“这句话，实际上，我也是因为看到这句话觉得有趣才写这篇文章的，并且我尽力用诙谐而嘲讽的笔风去写，显得更像一部史诗。 （于是我室友在路过我的位置看到我写这些的时候问我：”你在干嘛“，我连忙 alt + tab，有点羞耻的说） 二、讲述他在北大的”艰苦奋斗“ 他多次提到：当时北大会断电还是因为什么原因，导致他们要跑到路灯下面刷《吉米多维奇》，路灯很暗，而且外面很冷，常常要刷到很晚，总之是十分艰苦。 然而在我们都要开始同情他艰苦奋斗的时候，他都会重点强调：”那个时候刷题到很晚，在路灯下吃泡面，被一个同学过来把泡面抢走了”（带着浓重的怨妇口吻），然后大家就会顾不上想象他的艰苦，只觉得他的怨妇语调过于有趣，然后满堂哄笑。 三、吐槽中国数学教育 四、表示自己不归中国教育部管，可以为所欲为 他其实是南科大的访问学者，本身是巴西一个大学的教授，感觉这个人估计人生是有点传奇成分的，居然跑去巴西当教授了。 五、每次上课一瓶可乐 他的身材微胖，行动起来特别慵懒，关键还每次上课必带一瓶可乐，讲着讲着不时喝一口可乐。这简直是典型的肥宅形象，没有一点老师的样子，关键经常喝完一口可乐，就会开始说：”我不归中国教育部管“。然后整个教室就都笑得很开心。 我非常喜欢这种有个性、有棱角的老师，能够让人感受到他们身上的一些东西，能够被感染、被影响到。 六、多次谈到陪老婆逛街过程中想出问题的解法 记得第一次他说陪老婆逛街的时候想出问题的时候，我很惊讶：这个 b 这样子竟然有老婆（原谅我的不敬）。 带带大家刚 我们现在特别爱玩梗，纯纯的梗小鬼，常常因为单纯的玩梗让教室内充满活跃的气氛。而杨佳刚，和现在非常火的一位抽象明星神似——那就是当代 Hape 大王、抽象教主带带大师兄孙笑川。于是我们称他为带带大佳刚。 说杨佳刚很像带带大师兄，不仅仅是由于他的体型和长相，更多的还是由于线代这门课和孙笑川名言高度的匹配性以及杨佳刚有些行为和话语表现出其带带大师兄的气质所以我们觉得他像孙笑川。 孙笑川，有几个名言： nmsl，这个通常写作 nm$l，在线代课上常常用以此引申的你妈没了的变换 你把你闪现给我交了，在线代课上可以引申为交作业之类的语句，如”你把你作业给我交了” 你吼辣么大声干什么吗 抽象圣经，在线代课上指线代书 铁憨憨 杨佳刚的许多行为都显出他和孙笑川的高度相似性。 他经常把讲过的东西反复讲，然后让同学上去写回答，有的同学写不出来，他就表现的很无奈，下面的同学，经常是我和我的 Java project 队友——经常上课看恶俗维基的老哥亦博就会调侃：“你分儿没了”。两人相视一笑，甚是有趣。自几次之后，这种搭配又变形为“你卷（试卷）没了”，“宁分没了”之类的东西，每次都感觉是欢乐源泉。 有一次杨佳刚讲的正嗨，一脸吹牛逼吹到自己都信了的样子，突然停下来往门外走，众人一脸懵不知道他要干什么，于是甚为好奇。他走到教室门口把门关了起来，自言自语说道：“太吵了外面”。我和亦博相视一笑：“你吼辣么大声干什么吗！” 又是十分简单的好笑hhh。 线代每周都需要交作业，带带大佳刚每次其实都讲的蛮快的，但是要么是一开始浪要么是最后浪，反正就会莫名其妙的浪一波，最后告诉我们做哪些作业的时候会加上一句：“我这样完全就是为你们的安危考虑，真的是关心你们的安危”。这样的论调加上他铁憨憨的语调，瞬间显得很逗比很逗比，于是我和亦博会快乐的应和：“你把你作业给我交了”。 另附：梗小鬼现在爱玩的其它梗 坤坤梗 守护世界上最好的 kunkun 律师函警告！
